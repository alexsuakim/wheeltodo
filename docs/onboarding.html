ionstudioapps

Welcome to the team 👋

Follow these 6 steps to get your machine set up.
Takes about 10 minutes.

1

Create a GitHub account

You need this to access the team's code

Browser

Go to github.com/signup and create a free account.

💡 Use your personal email — you'll use this account for everything ionstudio-related.

Once you have an account, send your GitHub username to Sua so she can invite you to the team.

2

Accept the team invite

Check your email after Sua adds you

Browser

You'll get an email from GitHub with the subject "You've been invited to join ionstudioapps". Click the link and accept.

⚠️ If you don't see it, check your spam folder or go to github.com/orgs/ionstudioapps/invitation directly.
3

Install Claude Code

Your AI coding assistant

App

Download and install Claude Code from claude.ai/download.

Open it and sign in (or create an Anthropic account). You'll use this every day.

💡 Claude Code works best in the desktop app. Make sure to open it at least once before running the setup script.
4

Open Terminal

The command line app on your Mac

Terminal

Press ⌘ + Space, type Terminal, and hit Enter.

📸 Screenshot placeholder — Spotlight search showing Terminal

A window with a black or white background and a blinking cursor will open. That's the terminal — don't worry, you only need to paste one thing into it.

5

Run the setup script

This installs everything automatically

Terminal

Copy this line and paste it into your terminal, then press Enter:

bash <(curl -fsSL https://raw.githubusercontent.com/ionstudioapps/wheeltodo/main/scripts/setup.sh)

The script will:

  • Install developer tools (Homebrew, Node.js, GitHub CLI)
  • Log you into GitHub in the browser
  • Clone all the ionstudio repos onto your Mac
  • Install all dependencies
⚠️ It may ask for your Mac password at some point — that's normal. Nothing harmful is being installed.

The whole thing takes 3–5 minutes. When you see "Done!" you're all set.

6

Get the secrets file

API keys and credentials from Sua

Manual

Ask Sua for the rocky.env file and save it to:

~/Projects/rocky/.env
🔒 Never share this file or commit it to GitHub. It contains private API keys.
🚀

You're ready to go

Open Claude Code inside the wheeltodo project by running this in your terminal:

cd ~/Projects/wheeltodo && claude
